## Installation

- Clone the repository
   ```bash
   git clone https://github.com/luminahi/bouncing-balls.git
   ```
- Move to project
   ```bash
   cd bouncing-balls
   ```
- Install dependencies
   ```bash
   npm install
   ```
- Generate Javascript
  ```bash
  npm run build 
  ```

## Usage

- Run the application
   ```bash
   npm start
   ```
   the page will open in localhost, port 8080 by default.
