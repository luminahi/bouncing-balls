## Installation

- Clone the repository
   ```bash
   git clone https://github.com/luminahi/bouncing-balls.git
   ```
- Install dependencies
   ```bash
   npm install
   ```
- Generate Javascript
  ```bash
  npm run build 
  ```

## Usage

- run the application
   ```bash
   npx alive-server
   ```

the page will open
in localhost, port 8080 by default.
